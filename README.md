# Safenet

This is a work-in-progress. Don't use it yet!

## Project setup

- [Contracts](./contracts) Consensus contracts (Solidity & Foundry)
- [Validator](./validator) Validator service (Typescript & npm)

## Install dependencies

### Contracts
```
forge install
```

### Validator
```
npm install
```

## Run tests

Contracts
```
forge test
```

Validator
```
npm test
```

Integration
```
npm run test:integration
```
